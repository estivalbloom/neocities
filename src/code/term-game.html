Retro Computer Game Demo

Not much content to it; basically just a proof of concept for the look. Type "help" to list the allowed commands. If you're confused about what any of them do and just want to see how text on screen looks, try "lorem" or "color_party". To terminate a currently-running program, use ctrl+c. Using the "exit" command just kinda freezes the game right now.

VFX controls:
  • F1: Toggle screen warp
  • F2: Adjust screen warp radius
  • F3: Toggle chromatic aberration
  • F4: Adjust chromatic aberration distance
  • F5: Toggle scan lines
  • F6: Adjust scan line effect strength
  • F7: Toggle noise
  • F8: Adjust noise effect strength
  • F9: Reset effect configuration
Hold shift to adjust in the opposite direction
